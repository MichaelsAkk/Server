Список доступных команд: 
help - вывести справку по доступным командам
info - вывести в стандартный поток вывода информацию о коллекции (тип, дата инициализации, количество элементов и т.д.)
show - вывести в стандартный поток вывода все элементы коллекции в строковом представлении
add {element} - добавить новый элемент в коллекцию
update id {element} - обновить значение элемента коллекции, id которого равен заданному
remove_by_id id - удалить элемент из коллекции по его id
clear - очистить коллекцию
execute_script file_name - считать и исполнить скрипт из указанного файла. В скрипте содержатся команды в таком же виде, в котором их вводит
remove_greater {element} - удалить из коллекции все элементы, превышающие заданный (по полю numberOfRooms)
remove_lower {element} - удалить из коллекции все элементы, меньшие, чем заданный (по полю numberOfRooms)
history - вывести последние 8 команд (без их аргументов)
average_of_number_of_rooms - вывести среднее значение поля numberOfRooms для всех элементов коллекции
filter_by_number_of_rooms numberOfRooms - вывести элементы, значение поля numberOfRooms которых равно заданному
print_ascending - вывести элементы коллекции в порядке возрастания (по полю numberOfRooms)
